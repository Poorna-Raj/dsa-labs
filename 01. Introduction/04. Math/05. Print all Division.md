# Print all Division

## Brute Force Method

`input.txt`

```txt
12
```

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int main()
{
    int N;
    cin >> N;

    for (int i = 1; i <= N; i++)
    {
        if (N % i == 0)
        {
            cout << i << "  ";
        }
    }
}
```

`output.txt`

```txt
1  2  3  4  6  12
```

Time complexity = **O(N)**.

## Observed Method

![example](assets/factors%20of%20a%20number.png)

`demo.cpp`

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
    int x;
    cin >> x;
    vector<int> sortedFactors;

    for (int i = 1; i * i <= x; i++)
    {
        if (x % i == 0)
        {
            sortedFactors.push_back(i);

            if ((x / i) != i)
            {
                sortedFactors.push_back(x / i);
            }
        }
    }

    sort(sortedFactors.begin(), sortedFactors.end());

    for (auto it : sortedFactors)
    {
        cout << it << "  ";
    }
}
```

`output.txt`

```txt
1  2  3  4  6  12
```

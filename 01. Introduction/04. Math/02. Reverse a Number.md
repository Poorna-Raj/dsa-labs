# Reverse a Number

In this we will try to reverse a number.
`input.txt`

```txt
7789
```

`demo.cpp`

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main()
{
    int x, y, reverse;
    cin >> x;
    y = abs(x);
    reverse = 0;

    while (y > 0)
    {
        reverse = (reverse * 10) + y % 10;
        y /= 10;
    }

    cout << "Reverse = " << reverse << "\n";
}
```

`output.txt`

```txt
reverse = 9877
```

# Recursion

Recursion is when a **faction call itself** until a specific **condition become true**.

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

void func();

int main()
{
    func();
}

void func()
{
    cout << 1;
    func();
}
```

This program will print 1 to the console until it get's a **stack overflow**.

## Recursion with base condition

Base condition will determine when to stop the recursion.

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int count = 0;

void func()
{
    if (count == 2)
    {
        return;
    }
    cout << count << "\n";
    count++;
    func();
    cout << "After the func()" << "\n";
}

int main()
{
    func();
}
```

`output.txt`

```txt
0
1
After the func()
After the func()
```

![example](assets/recursion%20example%202.png)

There can be more than one base condition.

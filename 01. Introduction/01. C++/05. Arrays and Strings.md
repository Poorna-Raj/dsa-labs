# Arrays

Arrays are collection of data with same data type.

The data get stored `sequentially`.

Let's define an integer array with a size of 5,
`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int main()
{
    int arr[5];
}
```

![array definition](assets/array%20definition.png)

Let's add values,

`input.txt`

```txt
7
3
9
6
8
```

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int main()
{
    int arr[5];
    cin >> arr[0] >> arr[1] >> arr[2] >> arr[3] >> arr[4];

    arr[2] += 10;
    cout << arr[2];
}
```

`output.txt`

```txt
19
```

# 2D Arrays

These arrays are like matrices.

Let's take an example,

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int main()
{
    int arr[3][2];
}
```

# Strings

- Strings are basically arrays of characters.
- They also have the indexing capabilities that arrays have,

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int main()
{
    string name = "abbys";
    cout << name[0];
}
```

`output.txt`

```txt
a
```

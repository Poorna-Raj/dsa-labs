# Priority Queue

It is similar to a queue but the **larget value will stay on top**.

`demo.cpp`

```cpp
#include <iostream>
#include <queue>

using namespace std;

int main()
{
    priority_queue<int> que;

    que.push(4);
    que.push(10);
    que.push(6);
    que.emplace(4);

    cout << "Top is -> " << que.top() << endl;
}
```

`output.txt`

```txt
Top is -> 10
```

So when you pop the queue it will remove the 10 and make 6 as the front.

## Reverse the Property

Since the default priority queue store the maximum value as the top we can change it to store the minimum value.

`demo.cpp`

```cpp
#include <iostream>
#include <queue>
#include <vector>

using namespace std;

int main()
{
    priority_queue<int, vector<int>, greater<int>> que;

    que.push(4);
    que.push(10);
    que.push(6);
    que.emplace(4);

    cout << "Top is -> " << que.top() << endl;
}
```

`output.txt`

```txt
Top is -> 4
```

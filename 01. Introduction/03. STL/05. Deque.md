# Deque

This one also similar to list and vectors. Same methods as well.

## Declaration

`demo.cpp`

```cpp
#include <iostream>
#include <deque>

using namespace std;

int main()
{
    deque<int> dq;

    dq.push_front(77);
    dq.push_back(66);

    for (auto it : dq)
    {
        cout << it << " ";
    }
}
```

`output.txt`

```txt
77 66
```

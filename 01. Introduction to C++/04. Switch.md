# Switch Statements

- Based on a condition, `switch` selects one or more code blocks to be executed.

- `switch` executes the code blocks that matches an expression.

- `switch` is often used as a more readable alternative to many `if`...`else if`...`else` statements, especially when dealing with multiple possible values.

`input.txt`

```txt
1
```

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int main()
{
    int day;
    string dayName;
    cin >> day;

    switch (day)
    {
    case 1:
        dayName = "Monday";
        break;
    case 2:
        dayName = "Tuesday";
        break;
    case 3:
        dayName = "Wednesday";
        break;
    case 4:
        dayName = "Thursday";
        break;
    case 5:
        dayName = "Friday";
        break;
    case 6:
        dayName = "Saturday";
        break;
    case 7:
        dayName = "Sunday";
        break;
    default:
        dayName = "Invalid Day";
        break;
    }

    cout << "Happy " << dayName;
}
```

`output.txt`

```txt
Happy Monday
```

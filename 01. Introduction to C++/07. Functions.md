# Functions

- Functions are set of code which performs a specific task.
- Functions are used to modularised code.
- Functions are used to increase readability.
- Functions are used to execute same block of code multiple times.

## void functions

- It doesn't return anything.

`input.txt`

```txt
abbys
```

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

void printName(string &name);

int main()
{
    string name;
    cin >> name;
    printName(name);
}

void printName(string &name)
{
    cout << "Welcome " << name;
}
```

`output.txt`

```txt
Welcome abbys
```

## return functions

- These function will return a data type.

`input.txt`

```txt
3 5
```

`demo.cpp`

```cpp
#include <iostream>

using namespace std;

int sum(int num1, int num2);

int main()
{
    int num1, num2;
    cin >> num1 >> num2;
    cout << "The total = " << sum(num1, num2);
}

int sum(int num1, int num2)
{
    return num1 + num2;
}
```

`output.txt`

```txt
The total = 8
```
